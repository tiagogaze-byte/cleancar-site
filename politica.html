3. Dados Coletados

3.1 Dados dos Clientes da Plataforma (Você)

Coletamos as seguintes informações quando você se cadastra e utiliza nossa Plataforma:

  • Dados cadastrais: nome completo, e-mail, telefone, CPF/CNPJ
  • Dados de pagamento: informações de cobrança e pagamento processadas por gateway terceirizado
  • Dados de uso: logs de acesso, endereço IP, navegador, horários de uso
  • Configurações da conta: preferências de notificação, configurações de API

3.2 Dados dos Contatos (Leads) Gerenciados por Você

Nossa Plataforma processa, em nome do cliente, os seguintes dados dos contatos finais:

  • Número de telefone do WhatsApp
  • Nome (quando fornecido pelo contato)
  • Conteúdo das mensagens trocadas via WhatsApp Business API
  • Metadados de mensagens: horário de envio, entrega, leitura, status
  • Arquivos de mídia: imagens, áudios, vídeos e documentos
  • Dados de segmentação: tags, estágio no funil, origem do contato

IMPORTANTE: Você, como cliente da Plataforma, é o controlador de dados em relação aos seus contatos finais. Nossa Plataforma atua como operador de dados. Você é responsável por obter consentimento explícito, informar sobre o uso dos dados e respeitar os direitos dos titulares.

4. Finalidade do Tratamento de Dados

Utilizamos seus dados para prestação do serviço, cobrança, suporte técnico, melhoria da plataforma, comunicação e segurança.

Não utilizamos os dados dos contatos finais para nenhuma finalidade própria, marketing ou compartilhamento com terceiros não autorizados.

5. Base Legal para Tratamento de Dados (LGPD)

  • Execução de contrato (Art. 7º, V, LGPD)
  • Consentimento (Art. 7º, I, LGPD)
  • Cumprimento de obrigação legal (Art. 7º, II, LGPD)
  • Legítimo interesse (Art. 7º, IX, LGPD)

6. Compartilhamento de Dados
